Penjelasan tentang Aplikasi

Aplikasi ini digunakan untuk membantu kita dalam membuat suatu password yang unik dan kuat, dalam penggunaan aplikasi ini user hanya melakukan ini saja :

  • Memilih panjang range yang dibutuhkan untuk membuat password yang rumit
  • Melakukan pemilihan beberapa persyaratan untuk sebuah password seperti menggunakan kombinasi angka, huruf besar, huruf kecil, dan kombinasi simbol
  • Selanjutnyan jika sudah di isi maka mengkelil tombol yang ada di bawahnya untuk memulai membuat passwordnya
